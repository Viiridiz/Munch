## About This Project

> Munch helps you find the perfect recipe in seconds! Just input your ingredients, dietary preferences, or allergies, and our AI curates delicious options tailored to you. With a built-in health meter, cooking timer, and a seamless login system, Munch makes meal planning effortless and enjoyable.

## Key Features

* **Smart Recipe Search:** Find recipes by ingredients, dietary needs (e.g., vegan, gluten-free), or allergies.
* **AI-Powered Curation:** Uses the **OpenAI API** to generate and curate delicious, tailored recipe options.
* **Health Meter:** Provides a simple visual guide to the healthiness of a recipe.
* **Built-in Cooking Timer:** Keep track of your cooking times directly within the app.
* **User Accounts:** A seamless login system to save your preferences and favorite recipes.

## üèÅ Getting Started

Follow these instructions to get a local copy up and running.

### Prerequisites

* Node.js & npm (or yarn)
* An OpenAI API Key

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/munch-ai-assistant.git](https://github.com/your-username/munch-ai-assistant.git)
    cd munch-ai-assistant
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
3.  **Set up environment variables:**
    * Create a `.env.local` file in the root of the project.
    * Add your OpenAI API key:
        ```
        REACT_APP_OPENAI_API_KEY=your_api_key_here
        ```
4.  **Run the application:**
    ```bash
    npm start
    # or
    yarn start
    ```
5.  Open your browser and navigate to `http://localhost:3000`.
